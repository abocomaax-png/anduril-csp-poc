üîß Technical Details:

Vulnerable Endpoint:

Injected CSP Header:

Content-Security-Policy: frame-ancestors 'self' https://aplogviewer.anduril.dev

Root Cause:

The server dynamically builds the CSP header using the redirect_uri parameter without proper domain validation. An attacker can leverage this by:

  1. Registering a subdomain like aplogviewer.anduril.dev.attacker.com
  2. Or using the legitimate domain directly in their malicious page
  3. Embedding the OAuth authorization flow in an iframe
  4. Stealing authorization codes or performing clickjacking attacks
